
 ----------------------------------
       Astuces de navigation
 ----------------------------------

- Page principale (Coverflow) :
  Moins / Plus  =  Défilement rapide des jeux
  Moins hors de l'écran  =  Activer / Désactiver la barre d'icônes
  Plus hors de l'écran  =  Lettre (ou type de jeu) suivant(e)
  B + A sur un jeu  =  Lancement immédiat
  B + A sur l'icône source  =  Vue coverflow suivante
  B + A sur l'icône dvd  =  Lancement immédiat
  B + A sur l'icône paramètres  =  Paramètres globaux
  B + Moins / B + Plus  =  Sélectionner / lancer un jeu aléatoire
  B + Haut / B + Bas  =  Changer de nand émulée
  B + Gauche / B + Droite = Changer la musique
  Z  =  Capture d'écran
  
- Jeu sélectionné :
  A sur la boîte  =  Retourner la boîte
  Plus / Moins / Gauche / Droite sur boîte retournée  =  zoomer et bouger
  Moins hors de l'écran  =  Activer / Désactiver la barre d'icônes
  Plus hors de l'écran  =  Retourner la boîte

- Menu Home :
  Home  =  Raccourci vers la sortie par défaut

- Réglages des paramètres du Coverflow :
  A / 2 maintenus  =  Variation rapide des valeurs sélectionnées
  B + Moins / B + Plus  =  Page précédente / suivante

 ----------------------------------
         Mapping manettes
 ----------------------------------

- Manette classique :
  Bouton Moins / Plus  =  gâchette L / R

- Manette GameCube :
  Bouton Home  =  bouton Start
  Bouton Moins / Plus  =  gâchette L / R

- Guitare :
  Bouton A / B  =  Bouton Vert / Rouge
  Bouton 1 / 2  =  Bouton Bleu / Jaune

 ----------------------------------
         Emulation de nand
 ----------------------------------

- Jeux Wii (disques) :
  Off  =  Sauvegardes, DLC et Miis sur Wii
  Saves/DLC  =  Sauvegardes et DLC sur nand émulée
  Complète  =  Sauvegardes, DLC et Miis sur nand émulée
  Neek2o  =  Lancement du système menu neek2o sur nand émulée

- Chaînes Wii :
  Partielle  =  Configuration et Miis de la Wii
  Complète  =  Configuration et Miis de la nand émulée
  Neek2o  =  Lancement du jeu en mode neek2o
